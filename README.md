# Tutorial GitHub

Materiale didattico per l'introduzione alla piattaforma GitHub, destinato agli studenti del terzo anno del corso di informatica (TEPSIT).

## Contenuto

| File | Descrizione |
|------|-------------|
| `tutorial-github.html` | Tutorial completo in formato HTML con tema chiaro e diagrammi SVG |

## Argomenti trattati

Il tutorial copre i seguenti argomenti:

- **Cos'è GitHub** — storia, origine e utilizzi principali
- **Concetti fondamentali** — Repository (pubblico/privato), Branch, Clone, Commit, Push e Pull
- **GitHub Desktop** — download, installazione, configurazione e workflow pratico passo per passo
